Techniques

What makes it tick

01

Lenis Velocity API

Access real-time scroll velocity through Lenis's scroll event. Map the absolute value to visual intensity for reactive effects.

02

Canvas 2D Compositing

A fixed Canvas layer sits above the page with pointer-events: none. Speed lines are cleared and redrawn each rAF tick.

03

Three.js + ScrollTrigger

The WebGL hero canvas uses GSAP's ticker. Camera and mesh rotation are driven by scroll progress for seamless integration.

04

CSS Hue Shift

Gradient text hue is updated each frame from a ScrollTrigger onUpdate callback — no repaints, just CSS variable changes.
