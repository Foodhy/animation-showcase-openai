How It Works

This technique uses GSAP's TextPlugin combined with ScrollTrigger to animate numeric values from 0 to a target number while the user scrolls. The counter respects scrub: true, meaning the animation is directly linked to scroll position — scroll faster, the numbers update faster.

Key Features

  • Scroll-synced: Counter animation is tied directly to scroll progress using scrub
  • Decimal support: Handles percentages, decimals, and large numbers with thousand separators
  • Staggered reveal: Each card animates in sequence as you scroll past
  • Reduced motion: Counters are skipped if user prefers no motion
  • Performance optimized: Uses integer rendering to avoid excessive DOM updates

Use Cases

Perfect for:

  • Portfolio showcase pages (project counts, years of experience)
  • SaaS landing pages (user counts, uptime metrics)
  • Case studies (improvement percentages, revenue impact)
  • Company stats pages (customers served, transactions processed)

Integration Tips

To use this in your own projects:

  1. Import GSAP and TextPlugin: import { TextPlugin } from 'gsap/TextPlugin'
  2. Register the plugin: gsap.registerPlugin(TextPlugin)
  3. Use scrub: true to link animation to scroll position
  4. Customize formatting with formatUnits callback for decimals/prefixes
  5. Wrap in ScrollTrigger to trigger when card enters viewport
