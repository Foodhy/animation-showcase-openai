View Transitions API CSS Shared Elements

This demo uses the View Transitions API to animate between a card grid and a detail page. The card image and title have matching view-transition-name properties, allowing the browser to smoothly interpolate their position and size during the transition.

In browsers that don't support View Transitions, the content swaps instantly with a simple CSS crossfade fallback.
