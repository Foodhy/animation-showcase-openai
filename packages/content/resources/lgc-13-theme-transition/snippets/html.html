Theme Transition

Click the toggle below to switch themes with a circular clip-path wipe powered by the View Transitions API.

View Transitions API

The document.startViewTransition() method captures before/after snapshots and animates between them using CSS pseudo-elements.

Circular Wipe

A custom ::view-transition-new(root) animation uses clip-path: circle() expanding from the toggle button position.

CSS Custom Properties

Theme colors are defined as CSS variables on [data-theme]. The transition swaps theme by toggling this attribute.

The Art of Transitions

A well-crafted theme transition doesn't just swap colors — it tells a story. The circular wipe radiates from the point of interaction, creating a clear cause-and-effect relationship that feels natural and intentional.

Without the View Transitions API, achieving this effect would require complex overlay management, clipping containers, and careful z-index choreography. With it, the browser handles the heavy lifting.
