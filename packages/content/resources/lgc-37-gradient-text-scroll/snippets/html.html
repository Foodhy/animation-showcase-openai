Gradient Text Hue Shift

Text with animated gradient background that shifts hue and saturation as you scroll down the page. Pure CSS + GSAP ScrollTrigger.

How It Works

This effect uses CSS Custom Properties (CSS variables) to dynamically control the hue, saturation, and lightness of a gradient background applied to text.

As the user scrolls, GSAP ScrollTrigger updates the CSS variables in real-time, creating a smooth color shift that feels organic and modern.

The Technique

A heading element uses background: linear-gradient(...) with CSS variables for hue rotation. Each section has its own hue range.

The effect is simple but impactful: as you scroll past, the text color smoothly transitions through the color spectrum.

Perfect For

  • Hero sections: Draw attention with color-shifting headlines
  • Section headers: Visual rhythm and engagement
  • Feature callouts: Highlight key messages
  • Portfolio pages: Modern, eye-catching typography
  • Marketing sites: Brand color integration and movement

Advanced Variations

You can layer multiple hue shifts, adjust saturation for desaturation on scroll, or synchronize the shift with other animations on the page.

The gradient direction can also be linked to scroll velocity for even more dynamism.

Implementation

  1. Define CSS variables for hue: --hue: 0deg
  2. Apply gradient to text: background: linear-gradient(to right, hsl(var(--hue), 100%, 50%), ...)
  3. Use background-clip: text to show only the gradient on text
  4. Update --hue with ScrollTrigger's onUpdate callback
  5. Wrap in prefers-reduced-motion check for accessibility

Responsive & Accessible

The effect gracefully degrades on browsers without gradient text support. When prefers-reduced-motion is enabled, the gradient is static or removed entirely.
