Scroll Progress Indicators

Three indicator styles running simultaneously: top bar, circular SVG, and section dots. All driven by ScrollTrigger progress values.

Linear Bar

The top progress bar uses scaleX driven by overall page scroll progress. It's the simplest indicator — a single CSS transform.

Circular SVG

The circle uses SVG stroke-dashoffset to draw the ring progressively. The percentage text updates in real-time with scroll position.

Section Dots

Each dot represents a content section. The active dot highlights based on which section's ScrollTrigger is currently in view. Click a dot to jump to that section.

Combined Power

When used together, these indicators give users multiple spatial cues about their position in the content. Each serves a different cognitive need: linear for absolute progress, circular for at-a-glance percentage, dots for structural navigation.
